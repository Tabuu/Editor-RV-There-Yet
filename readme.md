# RV There Yet Editor
A basic editor for the game ["RV There Yet?"](https://store.steampowered.com/app/3949040/RV_There_Yet/)


## Example command
```ps
python3 .\main.py .\input_file.sav .\output_file.sav
```

## Save file location

### Windows
1. `%localappdata%\Ride\Saved\SaveGames`
